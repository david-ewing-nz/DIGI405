# Labour-NZ First Coalition

## About

This directory contains a corpus created using the [Conc](https://github.com/polsci/conc) Python library. 

## Corpus Information

Labour-NZ First Coalition (2017-2019) subcorpus, part of the Beehive (2014-2019) corpus compiled by Dr Geoff Ford. The corpus consists of speeches by New Zealand government ministers scraped from https://www.beehive.govt.nz/.
