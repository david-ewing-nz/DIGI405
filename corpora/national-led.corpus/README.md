# National-led

## About

This directory contains a corpus created using the [Conc](https://github.com/polsci/conc) Python library. 

## Corpus Information

National-led government (during the period 2014-2017) subcorpus, part of the Beehive (2014-2019) corpus compiled by Dr Geoff Ford. The corpus consists of speeches by New Zealand government ministers scraped from https://www.beehive.govt.nz/.
